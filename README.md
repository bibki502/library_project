# Library Project
1. Описание проекта (2-3 предложения)
   Консольное приложение для управления личной библиотекой, написанное на C++ с использованием объектно-ориентированного программирования.  
Позволяет добавлять книги и пользователей, выдавать и принимать книги, искать книги по ISBN и просматривать профили пользователей.  
Проект демонстрирует применение ООП: классы, объекты, инкапсуляцию, указатели и базовые методы STL.

3. Требования:
C++17 или выше
CMake 3.10 или выше
Среда: Windows (MSYS2 MinGW64) / Linux / macOS

3.Инструкции по сборке (шаг за шагом):

1. Открыть MSYS2 MinGW64 (или Linux/macOS терминал)
  
3. Перейти в корневую папку проекта:
cd /path/to/library_project

4.Создать папку для сборки и перейти в неё
mkdir build
cd build

5.Сгенерировать Makefile через CMake:
cmake ..
make

6.Запустить программу:
./labrary
